El siguiente proyecto hace referencia al desarrolo del siguiente sistema: 
Sistema de Inventario y ventas 

Este sistema tendrá los modulos de compras y ventas y por ello entonces se podrá administrar el inventario por medio de un control de inventario.
Dicha gestion de inventario administrará las entradas de productos (compras) salida de producto (ventas) y se podrán generar reportes de inventario por stock

Sistema desarrollo por aula 3DS
